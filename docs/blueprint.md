# **App Name**: Excel Workflow Automator

## Core Features:

- Multiple File Upload: Allow user to upload multiple excel files at once.
- Automated Logic Derivation: Leverage a tool to automatically derive spreadsheet processing logic, instead of expecting hard-coded instructions in code.
- Real-time Preview: Enable dynamic preview of the data transformations to occur.
- Easy Export: Provide one-click export to .xlsx format.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to convey a sense of calm and organization.
- Background color: Light gray (#F5F5F5), near-white to provide a neutral, clean backdrop.
- Accent color: Muted purple (#9370DB) to bring attention to CTAs and important UI elements.
- Body and headline font: 'PT Sans' (sans-serif) to combine a modern look and a little warmth and personality.
- Simple, monochrome icons to match the minimal and professional aesthetic.
- Clean, grid-based layout to facilitate ease of use, making the interface intuitive.
- Subtle transitions on data refresh/upload events to provide gentle feedback without distracting the user.
